# StatScrape
Web scraper for metrics data from StatWatch

## Why StatScrape?
- Statscrape pulls data from various metrics on Statwatch in order to create projections, compare to current goals, and update team members automatically.

## How does it work?
- Scrapes MTD and QTD Metrics data for each location,
- Calculate Projected Metrics for each location 
- Compare Projected data to current goals.
- Create XLSX with Sheets for Companies and Regions
- Emails Goal Summary to team, with a detailed spreadsheet attached.

### StatScrape supports these metrics, current and historical:
- Sales
- Customers
- Labor
- Weather

## Future Capabilities:
- Pull additional metrics (as needed) 
- Give user the ability to easily pick and choose desired functions.
